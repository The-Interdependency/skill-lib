# Meta Energy Theory Axioms

**Status**: strict root canon candidate.

This file uses **axiom** narrowly: a self-evident starting statement once its
terms are defined. If a claim requires a domain equation, empirical support,
implementation choice, or interface boundary, it is not a root axiom here.

Energy Theory is intended as a theory of all energy. Its current method is to
study how energy becomes legible through constrained relational architectures:
atoms, molecules, circuits, membranes, signs, conversations, institutions,
triads, heptagrams, and core-coupled computational systems.

The root set is deliberately small. The larger formula-backed material belongs
in [`LAW_FAMILIES.md`](LAW_FAMILIES.md) and [`OVERLAP_GRID.md`](OVERLAP_GRID.md).

## Root Axiom 0 — Distinction

```text
Without distinction, nothing is legible.
```

If no difference can be drawn, there is no measure, no relation, no boundary, no
identity, no direction, and no architecture available to thought or operation.

This is the root from which measurement and recognition begin.

## Root Axiom 1 — Permitted Relation

```text
Change occurs only through permitted relation.
```

Difference alone does not move. A transition requires some available relation:
path, coupling, medium, adjacency, permission, channel, boundary condition, or
geometry.

This axiom is the root behind flow laws. It does not yet say how much flow
occurs. That belongs to derived law-families.

## Root Axiom 2 — Coupled Emergence

```text
Relations can possess properties not possessed by isolated terms.
```

Distance is not in either point alone. A bond is not in either atom alone. A
circuit is not any one component. A conversation is not either speaker alone.

This is the root behind small network architecture and emergence.

## Root Axiom 3 — Ordered Transformation

```text
Order can change what a transformation is.
```

The same events in different order are not necessarily the same process. Timing,
phase, sequence, and provenance can be load-bearing.

This is the root behind fiqs, path dependence, phase, continuity, and semantic
provenance.

| Capacity | derived law-family | finite systems have finite tolerance relative to boundary conditions |

| Valence | derived law-family | compatibility follows from coupling, architecture, and threshold |

| Closure | derived law-family / postulate family | closure can stabilize, but triadic closure is a doctrine/postulate until formalized |
